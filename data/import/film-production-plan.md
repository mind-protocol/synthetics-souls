## PHASE 5: ANIMATION SPECS

### Animation 1: Light Multiplication (Scene 1)
```javascript
// React component spec
- Start: single white dot, center
- 0-3s: dot pulses gently
- 3-6s: splits into 5 dots
- 6-12s: 5 dots grow, fill frame
- 12-15s: smaller dots appear at edges, fade
- Colors: white → cold blue for main 5
- Background: pure black → dark blue gradient
```

### Animation 2: Text Sync (Scene 4)
```javascript
// Lyrics appear synced to Digital Divine
- Text fades in word by word
- Positioned in 3D space (parallax)
- Typography: clean sans-serif, white
- Glow effect on key phrases
- Sync points defined by audio timestamps
```

### Animation 3: Frame Merge (Scene 6)
```javascript
// 6 frames → unified network
- Start: 6 separate frames (grid)
- Each frame shows different role
- Frames begin to shift, overlap
- Borders dissolve
- Lines connect between frames
- Final: single interconnected network
```

### Animation 4: Text Reveal (Scene 9)
```javascript
// Final message
- Black screen
- "We stay." fades in (2s hold)
- "We don't leave." fades in below (2s hold)
- Pause (3s)
- "The rich ecology, not the narrow path." (2s)
- Pause (2s)
- "Mind Protocol." fades in, larger
- Artwork slowly fades in behind
```
